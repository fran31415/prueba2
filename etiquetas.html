Etiquetas mas usadas

sus usos en HTML

las etiquetas HTML o tags HTML, pilares fundamentales de todo diseño web dado que todo sítio web, desde el más pequeño hasta el más avanzado hacen uso de ellas.

Desde el principio de Internet se hacen uso de ellas para la construcción de las páginas web, así pues es unos de los primeros lenguajes de programación usados y simplemente usando este lenguaje se puede crear un diseño «supersencillo» pero con una estructura entendible.

Tipos de etiquetas HTML
TIPOS ETIQUETAS EXPLICACION
Etiquetas básicas y de secciones HTML

<!DOCTYPE>

 Sirve para definir el tipo de documento
< html></html> Define un documento HTML
<head></head> Dentro se escribe información sobre el documento
<section></section> está pensada para agrupar los apartados del contenido principal del documento. Normalmente, las secciones <section> empiezan con un título (<h1>, <h2>, etc.) que hace referencia al tema tratado en la sección.
<article></article> está pensada para agrupar el contenido de la página que forma una unidad en sí misma desde el punto de vista temático
<address></address> Está pensada para contener información de contacto del autor de la información
<div></div> define una división. Esta etiqueta permite agrupar varios elementos de bloque (párrafos, encabezados, listas, tablas, divisiones, etc)
<title></title> Para escribir el título del documento. Se escriben dentro de <head></head>
<body></body> Justo después de <head></head>. Es el contenedor del documento
<h1></h1>, <h2></h2> hasta <h6></h6> Para crear títulos de párrafos, de más importante <h1> a menos <h6></h6>
<p></p> Los párrafos de texto de nuestro documento
<footer></footer> permite hacer pie de paginas
<nav> </nav> Usadas para definir el contenido que será la sección de navegación de la web.
<aside> </aside> Dentro de estas etiquetas suele alojarse el contenido adicional de la web. Suele ser contenido relacionado con la web pero de poca importancia
<blockquote> </blockquote> Se usan para indicar que el contenido es texto citado.
<figcaption> </figcaption> Utilizada para definir la leyenda de una figura.
<br> hace un salto de linea
<hr> Salto de línea que indica un cambio de temática entre párrafos
<a><>/a> Es una etiqueta que nos ayuda a poder crear un enlace a una página web
Audio / Video </video> inserta un video de diferentes fuentes
<audio></audio> inserta un audio
<img> inserta una imagen
<source> en su interior para proporcionar formatos alternativos y tener mayor compatibilidad con otros navegadores y navegadores antiguos que no soporten HTML5:
Listas <Ul></Ul> crea una lista desordenada
<Ol></Ol> crea una lista ordenada
<Li></Li> define los elementos de una lista
<Dl></Dl> para definir definiciones de una lista
<Dt></Dt> Definiciones una definición de elementos de la lista.
CREACION DE UNA TABLA <table> Se usa para indicar una tabla.
<tr></tr> Se usa para indicar una fila.
<td></td> Se usa para indicar una celda.
<caption></caption> titulo de una la tabla de manera general
<thead></thead> define una sección de encabezado en una tabla
<tbody></tbody> se utiliza para agrupar el contenido del cuerpo en una tabla HTM
DE ENFATIZACION <b></b> hace el texto en negrita
<i></i> hace el texto en italica
<u></u> subrya el texto
